# Java图书管理系统 java951

## 项目概述

Java图书管理系统是一个基于SpringBoot和MySQL的图书管理解决方案，适用于图书馆或个人图书管理需求。

## 技术栈

- **后端：**
  - Java 1.8
  - SpringBoot
  - MyBatis
- **前端：**
  - Vue.js
  - HTML
  - CSS
- **数据库：**
  - MySQL 5.7/8.0
- **开发工具：**
  - IDEA
  - Eclipse

## 功能模块

### 系统角色

- **管理员：**
  - 首页
  - 个人中心
  - 用户管理
  - 图书分类管理
  - 图书信息管理
  - 图书借阅管理
  - 图书归还管理
  - 缴纳罚金管理
  - 留言板管理
  - 系统管理

- **用户：**
  - 首页
  - 个人中心
  - 图书借阅管理
  - 图书归还管理
  - 缴纳罚金管理
  - 我的收藏管理
  - 前台首页
  - 图书信息
  - 公告信息
  - 留言反馈
  - 后台管理

## 运行环境

- Java 1.8 或更高版本
- MySQL 5.7 或 8.0
- 现代浏览器

## 部署步骤

1. 准备开发环境（IDEA 或 Eclipse）
2. 导入项目代码
3. 配置数据库（MySQL 5.7/8.0）
4. 运行后端服务（SpringBoot）
5. 启动前端服务（Vue.js）
